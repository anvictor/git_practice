Это позволит вам создать удобную среду для программирования на html, less, css, php, js
больше не придется нажимать перезагрузку браузера. Все изменения отобразятся сразу после сохранения любого из файлов.

Вы распаковали архив. Его название gulp
Это настроенная в gulp среда программирования.

Скопируйте эту папку в новое место (это ваш новый проект)
Переименуйте папку gulp по новому.
изментите в файле package.json занчения

 "name": "put_there_new_name_of_you_new_project_and_rename_gulp_folder",

и другие значения, такие как автор, лицензия, версия и что еще хотите.

из консоли запустите 

npm install // это приведет к созданию большой папки с плагинами gulp

При желании переслать свой проект, нужно удалять эту папку.
На новом месте получатель всегда может запустить команду 

из консоли
npm install

все что описано в файле package.json само проинсталлируется.

Теперь время разработки:
запустите команду из консоли
gulp dev // режим разработчика 
или
gulp prod // если вам нужно собрать финишный продукт, что бы показывать его клиенту на хостинге.

В чем разница?
таск dev исполняет несколько тасков, которые собирают и показывают все налету в браузере.
Но в случае, если ваши файлы стилей устроены так, что главный файл style.less импортирует любые другие стилевые файлы командой 

@import "имя_другого_стилевого_фрагмента";

то в браузере при просмотре стилей элемента сохранится еще и указание из какого фрагмента взят стиль.
Просто, так его легче искать и менять.
Также таск dev делает из файлов js - ужасные фалы.js
и копирует файлы php
а так же  файл index.* (* - любое расширение. может.html, может.php)
далее запускает слежение за тем, 
было ли редактирование и обновляет браузер на лету.

таск prod тоже все сливает в те же папки, но без указания стилевых первоисточников и не запускает вочер и браузер-синхр

в папку ассетс кладите картинки он будет их копировать если добавились новые.
там есть следилка заих новизной.
Вы можете усовершенствовать эту среду по своему. 
например добавить удалялку картинок везде, если вы их удалили только из assets/img
Но это уже и так работает, если менять только файлы .less .js .php и index.*
